Add a new item

If an item is missing, or you wish to change add a variant you should add an item. If the value of the item needs to be changed, wait until the current stock is exhausted and then delete the old item before creating a new one.

Tax

Tax is calculated by comparing the total value of items available with the total credit of all the users in the store. As this is a non-profit system, the two totals must match so that when the last item is bought from the store, the remaining user credit is $0.00 or below.

Etiquette

It is good practice to make the purchase before collecting the item. You may also wish to put new items into the communal space before selling it to the store. Both of these things should be done as close as possible to one another in time to avoid confusion or incorrect stocktaking.
